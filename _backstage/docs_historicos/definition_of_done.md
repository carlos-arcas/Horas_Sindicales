# Definition of Done (DoD) técnico

Checklist obligatorio para cerrar un Pull Request:

- [ ] Tests pasan.
- [ ] Coverage >= baseline.
- [ ] No imports prohibidos.
- [ ] No `except Exception` nuevos.
- [ ] Logging consistente si toca operación crítica.
- [ ] Documentación actualizada si cambia comportamiento.
- [ ] `CHANGELOG` actualizado si aplica.
